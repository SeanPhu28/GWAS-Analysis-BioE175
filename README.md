# Exploring Shared Genetic Architecture Beyond European Genomes

**BE 175 — Machine Learning & Data-Driven Modeling in Bioengineering**  
UCLA Samueli School of Engineering · June 2026

**Authors:** Ethan Li, Matthew Pham, Sean Phu, Karin Fouad Fahim, Kody Seow

---

## Overview

This project extends Zhang et al. (2023) "FactorGo" by testing whether the
pleiotropic factors the model learned from 420,531 European individuals
generalize to African ancestry populations from Pan-UK Biobank.

We do not reimplement FactorGo. Instead, we take the published European
factor loadings from Zenodo and project African ancestry Z-scores onto
that factor space, measuring cross-validated R² as our evaluation metric.
tSVD and NMF fitted on African data serve as within-ancestry baselines.

**Core question:** Do the pleiotropic factors FactorGo learned from
Europeans explain the genetic signals seen in ~6,500 Africans?

## Course Methods

| Method | BE 175 Week | Role in This Project |
|--------|-------------|----------------------|
| tSVD | Week 3 | Model-free baseline on AFR data (paper's own comparison) |
| NMF | Week 3 | Additive-constrained baseline, probes directionality |
| R² | Week 2 | Fraction of AFR variance explained by EUR factor space |
| Cross-validation | Week 2 | 5-fold CV on variants — honest generalization estimate |
| Standardization | Week 3 | Required preprocessing before tSVD and NMF |

## Data Download

Raw data is not included due to file size (~15 GB total).
All data is publicly available with no registration required.

### 1. FactorGo European Results (Zenodo)

```bash
mkdir -p Data/raw/factorgo_european
curl -L -o Data/raw/factorgo_european/FactorGo_data_results.zip \
  "https://zenodo.org/api/records/7765048/files-archive"
cd Data/raw/factorgo_european
unzip FactorGo_data_results.zip
mkdir inner && mv FactorGo_data_results.zip inner/
unzip inner/FactorGo_data_results.zip -d .
cd ../../..
```

Required files after extraction:
- `Data/raw/factorgo_european/data/GWAS_Zscore.gz`
- `Data/raw/factorgo_european/results/FactorGo.Wm.tsv.gz`

### 2. Pan-UKB Variant QC File

```bash
curl -o Data/raw/panukb/full_variant_qc_metrics.txt.bgz \
  "https://pan-ukb-us-east-1.s3.amazonaws.com/sumstats_release/full_variant_qc_metrics.txt.bgz"
bgzip -d Data/raw/panukb/full_variant_qc_metrics.txt.bgz
```

## How to Run

### Option A — Python scripts (final polished pipeline)

Run from the project root in order:

```bash
python Code/preprocess.py   # ~15 min (rsID mapping reads 28M rows)
python Code/run_models.py   # ~2 min
python Code/analysis.py     # ~5 min
```

Each script checks that the previous step's outputs exist before running
and prints a clear error message if something is missing.

### Option B — Jupyter notebook (interactive exploration)

```bash
# Open in VS Code or Jupyter
Notebooks/analysis.ipynb
```

Run cells top to bottom. Each cell prints its result before you continue.
Cell 1 sets the working directory — make sure to run it first.

### Run Tests

```bash
pytest Tests/test_pipeline.py -v
```

All 34 tests should pass.
